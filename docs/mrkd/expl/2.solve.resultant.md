# Solving via resultant matrices


We load the needed packages and define the Macaulay solver class.
```@expl
using DynamicPolynomials, AlgebraicSolvers

MC = Macaulay()

```
 
We now define the variables and the polynomials:
```@expl
X = @polyvar x0 x1 x2 x3 
P=[ x0^2 + 2*x1^2 + 2*x2^2 + 2*x3^2  - x0,
 2*x0*x1 + 2*x1*x2 + 2*x2*x3  - x1,
 x1^2 + 2*x0*x2 + 2*x1*x3  - x2,
 x0 + 2*x1 + 2*x2 + 2*x3  - 1
]
```

We solve the system `P` using the Macaulay solver:

```julia
Xi, ms =  AlgebraicSolvers.solve(P, MC);
```
and we get 8 solutions (the columns of `Xi`):

### How does it work ?

First we compute the Macaulay matrix `R` of `P`:

```@raw html 
<pre><span style="color:red"> R, L = res_matrix(P,MC) </span></pre>
```

We perform a QR factorisation `F` of `N'` to obtain the indices `IB` of monomials forming a basis of the quotient.
Then we deduce from the factor `F.R` of the QR factorisation, the matrices of multiplication by the variables `X`:
```@raw html
<pre><span style="color:red"> M = _mult_matrices(F.R, L, IB, X)  </span></pre>
```
By Schur factorisation, we finally deduce the roots `Xi`, their multiplicities `ms` and the Schur eigenspaces `Z`:
```@raw html
<pre><span style="color:red">  Xi, ms, Z = schur_dcp(M)  </span></pre>
```
