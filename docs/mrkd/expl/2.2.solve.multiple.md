# Solving polynomial systems with multiple roots


## The roots

In this section, we illustrate the computation of roots from eigen computations, in the presence of multiple
roots. We consider the following system of two equations in two variables $x_1, x_2$:
```@example syspol2
using DynamicPolynomials, LinearAlgebra, AlgebraicSolvers, Groebner
x = (@polyvar x[1:2])[1]
P = [x[1]^2 + x[1] − x[2], x[2]^2 + x[1] − x[2]]
```
We compute a TNF using Grobner computation with exact coefficients (using the package `Grobner.jl`):
```@example syspol2
 GB = Grobner(
    Groebner.DegRevLex,      # function of the variables defining the monomial ordering
    Groebner.groebner,       # function for computing Grobner basis
    Groebner.normalform,     # function that computes the normal form of a polynomial
    Groebner.quotient_basis  # function that computes a basis of the quotient algebra
)
N, L = tnf(P, GB)
round.(N, digits=6)
```
	
Let $B=[1, x_1, x_2, x_1 x_2]$ be the first $4$ elements of $L$. Then we verify that $L= B^+$:
```@example syspol2
tuple(L...) # hide
```
The matrix `N` represents the TNF map $N: W \longrightarrow \mathbb{R}^4$ where $W = \langle L \rangle$. It is computed as follows:

  - first we compute a Grobner basis `G`  of `P`, using the function ` Groebner.groebner` with the monomial ordering `Groebner.DegRevLex(variables(P))`. The coefficients of the polynomials in this computation are rational $\in \mathbb{Q}$;
  - then we compute a basis `B` of the quotient by the ideal $I=(P_1, P_2)$ using `Groebner.quotient_basis`;
  - Finally we compute the reduction of the monomials in $L$ by `G` (using `Groebner.normalform,`) and decompose the remainder in the basis `B` using `Float64` numbers. These gives the columns of `N`.
  
We compute the multiplication matrices by the variables $x_1, x_2$ in the basis $B$, using the TNF `N` indexed by the monomials $L$:
```@example syspol2
M = mult_matrices(N, L , collect(1:4), x)
round.(M[1], digits=6)
```

To obtain the roots and their multiplicities, we perform a joint Schur factorization:
```@example syspol2
Xi, ms = schur_dcp(M)
round.(Xi, digits=6)
```
The solutions are the columns of `Xi`.
The corresponding multiplicities are
```@example syspol2
length.(ms)'
```
and the Schur factor is `Z`.

All this computation can be done with the following command:  
```@example syspol2
Xi, ms = AlgebraicSolvers.solve(P,  GB)
round.(Xi, digits=6)
```

## The inverse system

We now analyze the inverse system of the multiple point $\xi_0=[0,0]$:
```@example syspol2
D, B = invsys(P,[0,0])
D
```
These series represent a basis of the inverse system of $I=(P_1, P_2)$ at the point $[0,0]$. They are described in terms the dual basis $dx_1^{\alpha_1}dx_2^{\alpha_2}$ of the monomial basis $\mathbf{x}^{\alpha}=x_1^{\alpha_1}x_2^{\alpha_2}$. 

A basis of the local algebra $\mathcal A_{\xi_0}$ is `B`:
```@example syspol2
tuple(B...) # hide
```
We verify that `B` is dual to `D`:
```@example syspol2
[dot(l,b) for l in D, b in B]
```
This inverse system is generated by `D[3]`, since
```@example syspol2
x[1]^2*D[3], x[2]^2*D[3], x[1]*x[2]*D[3], x[1]*D[3], x[2]*D[3], D[3]
```
span the same space of series as `D`.
