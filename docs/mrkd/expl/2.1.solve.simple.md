# Solving polynomial systems with simple roots

In this section, we illustrate the computation of roots from eigen computations. We consider the following system of two equations in two variables $x_1, x_2$:

```@example syspol1
using DynamicPolynomials, LinearAlgebra, AlgebraicSolvers
x = (@polyvar x[1:2])[1]

P = [x[1]^2 + 11*x[1]*x[2]+2*x[1]-14*x[2] , x[2]^2 + 2*x[1]*x[2] -3*x[2]]
```

We compute a TNF from the Macaulay resultant-type matrix:
```@example syspol1
R, L = res_matrix(P, Macaulay())
round.(R, digits=6)
```
The rows represent the monomial multiples of degree $\le 3$ of the equations `P[i]`. 
It is a matrix of size $6\times 10$, which columns are indexed by the monomials in `L`:
```@example syspol1
tuple(L...)  #hide
```
We deduce the associated TNF `N`, as the kernel of `R`:
```@example syspol1
N, _ = tnf(R)
round.(N, digits=6)
```
The matrix `N` represents the TNF map $N: W \longrightarrow \mathbb{R}^4$ where $W = \langle L \rangle$. We select a family $B=[1, x_1, x_2, x_1, x_2]$, which forms a basis of $\mathcal A=R/I$ (where $I=(P_1, P_2)$):
```@example syspol1
IB = [1,3,2,5]; B = L[IB]
tuple(B...)  # hide
```
We compute the multiplication matrices by the variables $x_1, x_2$ in the basis $B$, using the TNF `N` indexed by the monomials $L$:
```@example syspol1
M = mult_matrices(N, L , IB, x)
round.(M[1], digits=6)
```
The columns of $M_1$ represents the multiplication of the elements of $B$ by $x_1$ modulo $I$, written in the basis of $\mathcal A=R/I$, as we can check easily on the columns 1, 3 representing $x_1\times 1$, $x_1\times x_2$.
Here, we have used Proposition $\ref{...}$, since $B^+ \subset W$ and `N[:,IB]` is invertible.
Its eigenvalues are:
```@example syspol1
round.(eigvals(M[1])',digits=6)
```

The eigenvectors of the transposed multiplication operators, scaled to have their first corrdinates equal to $1$ are:
```@example syspol1
E = eigvecs(M[1]'); for i in 1:size(E,2) E[:,i]/= E[1,i] end; E
```
By Proposition $\ref{...}$, they represent the evaluation at the roots of $I$, that is, the vector of monomial $B$, evaluated at the roots. 
This let us see what are the roots, since $B=[1, x_1, x_2, x_1, x_2]$ contains $x_1, x_2$. 

```@example syspol1
E[[2,3],:]
```
We verify that we obtain the same points (up to a permutation) by a joint diagonalization of $M_1, M_2$:
```@example syspol1
Xi, _ = diagonalization(M)
round.(Xi,digits=6)
```
The $4$ solutions of the system are the columns of `Xi`. They are of multiplicity $1$ (since the eigenvalues are simple).


All this computation can be done with the following command:  
```@example syspol1
Xi, ms = solve(P,  Macaulay())
Xi
```
