# Series

Series are sequences of values indexed by monomials. 

```math
    \sigma = \sum_{\alpha \in A \subset \mathbb{N}^n } \sigma_{\alpha} \mathbf{y}_1^{\alpha_1} \cdots  \mathbf{y}_n^{\alpha_n}
```

They represent linear functionals on polynomials, or dual elements of the polynomials.

```math

    \sigma: p = \sum_{\alpha}  p_{\alpha}  \mathbf{x}^{\alpha} \mapstop \langle \sigma , p \rangle = \sum_{\alpha} \sigma_{\alpha}\, p_{\alpha}

```

They are implemented as dictionaries between monomials and coefficients
