# Eigensolvers

Recovering the roots from operators of multiplication is done by join diagonalization 
when the roots are simple.

```@docs
AlgebraicSolvers.diagonalization
AlgebraicSolvers.eigdiag
```

When the roots are multiple, one can use the joint Schur factorization:
```@docs
AlgebraicSolvers.schur_dcp
```

The clustering of eigenvalues is done with the following function:
```@docs
AlgebraicSolvers.multiplicities
```
