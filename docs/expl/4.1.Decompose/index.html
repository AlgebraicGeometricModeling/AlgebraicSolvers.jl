Decomposition of series

using DynamicPolynomials, AlgebraicSolvers, LinearAlgebra
X = @polyvar x1 x2 x3 monomial_order = Graded{Reverse{LexOrder}}
(x1, x2, x3)

We want to find a sparse representation of the following series known up to degree 3:

sigma = dual(6.0 + 4.0*x1 + 15.0*x2 + 6.0*x3 + 6.0*x1^2 + 20.0*x1*x2 + 4.0*x1*x3 + 43.0*x2^2 + 15.0*x2*x3 + 6.0*x3^2 - 26.0*x1^3 + 30.0*x1^2*x2 + 6.0*x1^2*x3 + 72.0*x1*x2^2 + 20.0*x1*x2*x3 + 4.0*x1*x3^2 + 129.0*x2^3 + 43.0*x2^2*x3 + 15.0*x2*x3^2 + 6.0*x3^3)
6.0 + 4.0dx1 + 15.0dx2 + 6.0dx3 + 6.0dx1^2 + 20.0dx1*dx2 + 4.0dx1*dx3 + 43.0dx2^2 + 15.0dx2*dx3 + 6.0dx3^2 - 26.0dx1^3 + 30.0dx1^2dx2 + 6.0dx1^2dx3 + 72.0dx1*dx2^2 + 20.0dx1*dx2*dx3 + 4.0dx1*dx3^2 + 129.0dx2^3 + 43.0dx2^2dx3 + 15.0dx2*dx3^2 + 6.0dx3^3

We construct the Hankel matrix associated to $\sigma$ in degree $(1,2)$

L1 = monomials(X,0:1)
L2 = monomials(X,0:2)
H = hankel(sigma,L1,L2)
4×10 Matrix{Float64}:
  6.0   4.0  15.0   6.0    6.0  20.0   4.0   43.0  15.0   6.0
  4.0   6.0  20.0   4.0  -26.0  30.0   6.0   72.0  20.0   4.0
 15.0  20.0  43.0  15.0   30.0  72.0  20.0  129.0  43.0  15.0
  6.0   4.0  15.0   6.0    6.0  20.0   4.0   43.0  15.0   6.0

The rank of $H_{\sigma}$ will give us the dimension of `\mathcal{A}_{\\sigma}`.

rank(H)
3

We expect to find 3 exponential terms in the decomposition 

B0 = L1[1:3]
H0 = hankel(sigma, B0, B0)
H1 = hankel(sigma, B0, B0*x1)
H2 = hankel(sigma, B0, B0*x2)
H3 = hankel(sigma, B0, B0*x3);
Hl  = [H1,H2,H3]
M = [ H0^(-1)*Hl[i] for i in 1:3 ]
3-element Vector{Matrix{Float64}}:
 [1.1102230246251565e-16 9.142857142857146 -0.5714285714285718; 1.0 3.857142857142857 1.571428571428572; -1.1102230246251565e-16 -4.2857142857142865 1.1428571428571432]
 [1.1102230246251565e-16 -0.5714285714285718 -3.7142857142857157; 0.0 1.571428571428572 -0.2857142857142865; 0.9999999999999998 1.1428571428571432 4.42857142857143]
 [1.0 1.1102230246251565e-16 1.1102230246251565e-16; 2.220446049250313e-16 1.0 0.0; 0.0 -1.1102230246251565e-16 0.9999999999999998]

Then we compute a joint diagonalization of the pencil M:

Xi, E = diagonalization(M)

This gives us the 3 points Xi (one per column):

Xi
3×3 Matrix{Float64}:
 4.0  2.0  -1.0
 2.0  3.0   1.0
 1.0  1.0   1.0

and the common eigenvectors E. We compute the weights of the decomposition as 

w = diag(E'*H0*E)
3-element Vector{Float64}:
 -1.2894736842105203
  3.9516129032258096
  1.4202898550724634

We deduce the sum of exponential decomposition:

\[\sigma = \sum_{i=1}^{3} w_{i} \mathbf{e}_{\Xi[:,i]}\]
