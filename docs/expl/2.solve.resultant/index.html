Solving via resultant matrices

We load the needed packages and define the Macaulay solver class.

using DynamicPolynomials, AlgebraicSolvers
MC = Macaulay()

We now define the variables and the polynomials:

X = @polyvar x0 x1 x2 x3
P=[ x0^2 + 2*x1^2 + 2*x2^2 + 2*x3^2  - x0,
 2*x0*x1 + 2*x1*x2 + 2*x2*x3  - x1,
 x1^2 + 2*x0*x2 + 2*x1*x3  - x2,
 x0 + 2*x1 + 2*x2 + 2*x3  - 1
]
4-element Vector{DynamicPolynomials.Polynomial{DynamicPolynomials.Commutative{DynamicPolynomials.CreationOrder}, Graded{LexOrder}, Int64}}:
 -x0 + 2x3² + 2x2² + 2x1² + x0²
 -x1 + 2x2x3 + 2x1x2 + 2x0x1
 -x2 + 2x1x3 + x1² + 2x0x2
 -1 + 2x3 + 2x2 + 2x1 + x0

We solve the system P using the Macaulay solver:

Xi, ms =  AlgebraicSolvers.solve(P, MC)

and we get 8 solutions (the columns of Xi):

How does it work ?

First we compute the Macaulay matrix R of P:

 R, L = res_matrix(P,MC)

The resultant matrix R is a sparse matrix, which columns are indexed by the list of monomials L, and which rows represent monomial multiples of the polynomials P:

  N, _ = LinearAlgebra.nullspace(R)

We perform a QR factorisation F of N' to obtain the indices IB of monomials forming a basis of the quotient. Then we deduce from the factor F.R of the QR factorisation, the matrices of multiplication by the variables X:

 M = _mult_matrices(F.R, L, IB, X)

By Schur factorisation, we finally deduce the roots Xi, their multiplicities ms and the Schur eigenspaces Z:

  Xi, ms, Z = schur_dcp(M)
